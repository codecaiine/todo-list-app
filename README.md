## Get Started

Please get your browser update.
To get a local copy up and running follow these simple steps.

Run the command below from the Terminal:

      git clone <URL> in your terminal

	  cd todo-list-app

	  npm install

	  npm run build

	  npm run start

	  npx hint

	  npx stylelint "**/*.{css,scss}"
